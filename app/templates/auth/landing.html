Comprehensive Feature Set

Every feature you need to streamline your trade operations and ensure compliance

Customs Processing Features
  • Auto-capture BL/AWB, invoice & packing list data (OCR + validation)
  • HS Code recommendation with rule engine & justification trail
  • One-click customs declarations with duty/tax calculation
  • Live milestone tracking across carriers, brokers & customs portals
  • Compliance checks against tariff changes, sanctions & documentation rules
  • Native integrations with ERP/WMS and shipping lines via secure APIs
Shipment Tracking Features
  • Live container/BL/AWB tracking with vessel/flight, ETD/ETA updates
  • Customs visibility: manifest → assessment → duty payment → release
  • Clearance milestones & alerts: document check, inspection holds, gate-in/out
  • Demurrage & detention counters with configurable thresholds
  • Delivery Order (DO) status, trucking assignment, last-mile GPS, POD capture
  • Exception dashboard with SLA timers, delay reasons, exportable audit trail
Document & Compliance
  • AI-driven document verification (OCR, field validation, tamper checks)
  • Continuous expiry monitoring with rolling calendars & alerts
  • Smart notifications via email/SMS/in-app with follow-up rules
  • Escalation matrix with SLA timers for compliance teams
  • Renewal workflows: prefilled forms, e-signature, authority portals
  • Audit-ready trails & dashboards with PDF/Excel export
Communication Hub
  • Unified in-app chat with @mentions, file sharing, threads
  • Built-in email & SMS with templates and scheduling
  • Rules-based alerts on milestones, exceptions, expiries
  • Automated escalation matrix with SLA timers
  • Case/ticketing from alerts with AI summaries
  • End-to-end visibility with unified inbox & webhooks
